# XQNT Coin

The official launch website for **XQNT Coin — X Quantum Network Token**.

XQNT is currently in its community and planning phase. This website does not
offer tokens for sale, collect wallet details, or display an unofficial contract
address.

## Local development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run build
npm test
```

Brand and launch details are centralized in `app/site-config.ts`.
